# IPL-Auction-Analysis

This focuses on analyzing IPL Auction data to gain insights into player performance, team composition, and auction trends. The analysis is performed using Microsoft Excel, featuring data cleaning, pivot tables, charts, and interactive dashboards.

The goal is to present a data-driven summary of how teams spent on players, identify top buys, and highlight trends in player categories (batsmen, bowlers, all-rounders, etc.).
